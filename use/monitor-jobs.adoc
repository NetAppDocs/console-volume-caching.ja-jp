= 監査ページでボリューム キャッシュ ジョブを監視する
:hardbreaks:
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetAppコンソールの監査ページを使用して、すべてのボリューム キャッシュ ジョブを監視し、その進行状況を確認できます。

.手順
. NetAppコンソール メニューから、*管理* > *監査* を選択します。
. 必要に応じて、時間、サービス、アクション、エージェント、リソース、ユーザー、またはステータスでフィルタリングします。
. すべてのキャッシュと操作を調べます。
